■１ はじめに
このプログラムを使うと、RX62NのUSBブートモードと通信し、
内蔵ROMにプログラムを書き込んだり、消去したり、ベリファイを行うことができます。
実行ファイルは、rxprog.exeです。

■２ デバイスドライバについて
RX62NのUSBブートモードと通信するため、汎用のデバイスドライバである、
libusbを使用しています。ドライバをインストールするためには、\driverフォルダにある
rxusbbootalt.infを指定してください。

■３ 使用方法
① RX62NのMD0ピンをジャンパ等Lレベルにし、リセットボタンを押します。

② MS-DOSプロンプトで、以下のコマンドを入力します。
 rxprog -p filename.mot  内蔵ROMへの書き込み(消去とベリファイも行う)
 rxprog -b               ブランクチェック
 rxprog -v filename.mot  ファイルの内容とROMの内容とのベリファイ
 rxprog -e               内蔵ROM(ユーザマット)の全消去
 rxprog -s               RX62N書き込み情報の表示

③ 書き込みの場合、以下のように画面が表示され、実行されます。

④ RX62NのMD0ピンをHレベルに戻し、リセットボタンを押します。
   ユーザプログラムが開始します。

■４ このプログラムで、まだできないこと
・IDコードプロテクトがかけられているROMのプロテクト解除
・消去ブロックにロックビットが設定されている場合の取り扱い
・ユーザブートマットの取り扱い
・内蔵ROMのリードバック
・インテル形式HEXファイル、ELFファイル、生のBinaryファイルからの書き込み
